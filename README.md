Building a new site for Alpha (thinkalpha.org)!

# getting started
install `npm`.
https://www.npmjs.com/get-npm

install the `gatsby` command line tool
run `npm install --global gatsby-cli`

start gatsby
run `gatsby develop`

open in your browser
http://localhost:8000/
